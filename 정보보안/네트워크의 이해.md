## 네트워크의 개념

### 네트워크

* 서로 떨어져 있는 여러 대의 컴퓨터 시스템이나 정보 회선을 이용하여 연결함으로써 정보를 교환하도록 구성한 통신망

- 정보교환, 자원 공유, 원격 제어 제공

### 네트워크의 분류

1. 개인 영역 네트워크
2. 근거리 네트워크
3. 광역 네트워크
4. 인터넷
5. 이동 통신망

#### 개인영역 네트워크(PAN)

휴대용 정보 단말기 등을 이용하여 필요한 정보를 처리할 수 있도록 구성한 네트워크

- 근거리 네트워크보다 소규모
- 블루투스, 근거리 무선통신 및 USB를 이용한 정보 기기들 간의 통신 기술을 이용

#### 근거리 네트워크(LAN)

여러 대의 컴퓨터와 정보 기기가 전용 통신 회선을 이용하여 연결되어 있는 네트워크

* 사무실, 학교, 빌딩 등과 같이 한정된 지역 내에서 정보를 교환하는 소규모 통신망
* 고속통신 가능하고 확장이 간편하며, 통신 오류율이 낮음

#### 광역 네트워크(WAN)

둘 이상의 LAN이 넓은 지역에 걸쳐 연결되어 있는 네트워크(지역과 지역, 국가와 국가, 대륙과 대륙)

- 비교정 넓은 지역을 지원하는 통신망
- 둘 이상의 LAN을 연결하기 위해서 라우터(router)라는 네트워크 장치 이용
  - 라우터(router) : 하나의 LAN과 또 다른 네트워크를 연결

#### 인터넷

전 세계에 있는 컴퓨터들을 서로 연결한 네트워크

* 전 세계에 있는 통신망과 통신망을 연동해 놓은 네트워크의 집합
* 전 세계 컴퓨터에 있는 정보를 공유 및 교환
* 전자우편(e-mail), 텔넷(Telnet), 파일 전송(FTP), 월드와이드 웹(WWW)
  * 텔넷(Telnet) : 물리적으로 떨어져 있는 컴퓨터를 원거리에서 온라인으로 연결하여 사용하는 원격 접속
  * 파일전송(FTP) : 인터넷상의 한 컴퓨터에서 다른 컴퓨터로 파일을 전송하는 서비스

#### 이동 통신(mobile telecommunication)

열차, 선박 등 이동체 상호 간 및 이동체와 이동체 외부와의 통신을 목적으로 하는 통신망

- 이동체의 경로 설정, 위치 추적, 기지국 설정 등 이동체를 대상으로 한 특유의 기능이 필요
- 이동 전화, 열차 젼화, 선박 전화, 무선 데이터 통신, 저궤도 위성 통신

## 네트워크 구성 요소

구성 요소는 크게 **하드웨어** 와 **소프트웨어**로 나눌수 있다

### 하드웨어

1. 송 · 수신 장치
2. 전송 장치
3. 전송 매체
4. 네트워크 제어 장치

#### 송 · 수신 장치

- **정보 처리**를 담당하는 단말 장치로, 통신을 이용할 수 있도록 통신 회선을 연결하여 네트어크 제어 장치에 접속되는 장치

- 단말 장치는 **원격지로부터 데이터의 입출력을 가능**하게 하는데 주로 컴퓨터가 이용

#### 전송 장치

- 송 · 수신 장치나 네트워크 제어 장치로부터 받은 신호를 통신 회선으로 보낼 수 있는 형태의 신호로 바꾸어 주는 장치

#### 전송 매체

- 전송매체인 통신 회선(유선 회선, 무선 회선)은 단말 장치에서 생성되는 신호를 상대측에 전송하는 역할
  - 유선 회선 : 꼬임선, 비차폐(UTP) 케이블, 동축 케이블, 광섬유 케이블 등
  - 무선 회선 : 라디오파, 마이크로파 등

#### 네트워크 제어 장치

- 통신 회선을 통해서 전송되어 온 정보를 처리하거나 저장
- 송신 장치와 수신 장치를 연결하거나 다른 네트워크를 선택하는 등 네트워크 관리 기능

### 소프트웨어

1. 통신규약
2. 통신 소프트웨어

#### 통신 규약

- 사용하는 언어와 통신 방식 등의 규격을 일치시키는 약속
- TCP/IP, FTP, HTTP
  - TCP/IP : 인터넷을 통한 모든 전송방식을 규정한 프로토콜의 집합
  - FTP : 인터넷상의 한 컴퓨터에서 다른 컴퓨터로 파일을 전송하는 서비스
  - HTTP : 인터넷에서, 웹 서버와 사용자의 인터넷 브라우저 사이에 문서를 전송하기 위해 사용되는 통신 규약

#### 통신 소프트웨어

- PC와 대형 컴퓨터 혹은 PC와 PC를 연결해주는 프로그램
- 웹 브라우저, FTP 프로그램. 전자 우편 프로그램, 메신저 프로그램, 텔넷(Telnet) 프로그램

### 프로토콜의 개념

통신 개체(entity)들 간에 신뢰성 , 효율성 및 안전성 있게 정보를 주고받을 수 있게 하기 위해서 규정한 여러 가지 통신 규칙과 방법

- 통신 개체 : 응용 프로그램, 전자우편 소프트웨어, 브라우저 등

* 서로 다른 통신 개체들끼리 통신을 원활하게 이루어지기 위해서는 개체들끼리 접속, 전달방식, 전송할 데이터의 형식, 오류의 검출 방식 등을 **표준화**하여 일치
* **상호 동일한** 프로토콜을 채택하여 통신망을 구축

### 프로토콜의 기본 요소

1. 구문
2. 의미
3. 타이밍

#### 구문(systax)

데이터의 구조나 형식을 가리키는 것

- 데이터의 형식, 부호화, 신호 크기 등을 포함
- 데이터가 어떤 순서로 표현되는지를 포함

#### 의미(semantics)

비트에서 각 부분의 뜻을 가리키는 것

- 전송 정보 패턴에 대한 해석, 해석에 따른 전송 제어, 오류 수정에 대한 제어 규정을 포함

#### 타이밍(timing)

- '얼마나 빨리 전송할 것인가'와 '언제 데이터를 전송해야 할 것인가'라는 두 가지 특성을 포함
- 두 개체 간의 통신 속도를 조정하거나 메시지의 전송 시간 및 순서에 대한 조정을 포함

### 전송 매체의 종류와 특징

전송 매체는 전기적인 신호나 빛의 형태 또는 전자파로 변환된 송신 신호가 수신 측에 전달되도록 통로 역할을 담당

1. 유선 전송 매체
2. 무선 전송 매체

### 유선 전송 매체

두 지점 간의 물리적인 전송로를 구성하여 데이터를 주고 받음

1. 꼬임선
2. 동축 케이블
3. 광섬유 케이블

#### 꼬임선

* 동선의 외부를 비닐로만 포장한 비차폐(UTP) 케이블 / 외부 비닐의 안쪽을 은박지로 차폐한 차폐(STP) 케이블
* 전송 속도가 느리지만 비용이 저렴하여 전화선이나 건물 내의 통신 회선등에 사용

#### 동축 케이블

- 저주파에서 수십 MHz의 고주파까지 넓은 주파수 범위의 전기 신호를 전송
- 장거리 통신망, 케이블 텔레비전 등에 활용

#### 광섬유 케이블

- 데이터를 빛에 의해 전송하므로 전기적인 신호의 간섭이 적음
- 전송 대역폭이 넓고 전송 속도가 동축 케이블이나 꼬임선에 비해 매우 빠름

### 무선 전송 매체

1. 라디오파
2. 마이크로파

#### 라디오파

다방향성 전파로서 안테나에서 발사한 전자파는 모든 방향으로 전파

* 라디오, 텔레비전 방송, 무선 전화

#### 마이크로파

통신 선로를 설치하기 힘든 지역에 사용하며 직접파를 이ㅛㅇ하므로 높은 지역에 안테나를 설치하여 전파를 전송

1. 이동통신
   - 고정된 위치가 아닌 장소에서 이동 중에 무선으로 통신
2. 위성 통신
   - 대기권 밖의 상공에 쏘아 올린 인공위성으로 하여금 통신을 중계하게 하는 통신 방법
   - 대륙 간 통신 또는 육해 간 통신

### 네트워크 장비의 종류와 기능

1. 랜 카드
2. 비차페(UTP) 케이블
3. 허브
4. 라우터

#### 랜 카드(네트워크 인터페이스 카드)

통신 케이블과 컴퓨터를 연결하여 데이터를 송 · 수신을 제어하는 장비

컴퓨터를 **근거리 네트워크**에 연결시키는 장비

#### 비차폐(UTP) 케이블

두개의 구리선을 꼬아서 만든 여러 개의 비차폐 케이블을 비닐로 감싸서 전류가 통하지 않게 만든 전송 매체

**컴퓨터**와 **네트워크 장비**를 연결하는 데 주로 쓰이는 통신 케이블

#### 허브

사무실이나 소규모 공간에서 컴퓨터들을 연결하는 네트워크 장비

#### 라우터

근거리 네트워크와 광역 네트워크처럼 서로 다른 네트워크를 상호 연결하는 네트워크 장비
