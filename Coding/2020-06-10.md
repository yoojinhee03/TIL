### 가운데 글자 가져오기



> https://programmers.co.kr/learn/courses/30/lessons/12903#



| s       | return |
| ------- | ------ |
| "abcde" | "c"    |
| "qwer"  | "we"   |



```java
class Solution {
    public String solution(String s) {
        String answer = "";
        
        if(s.length()%2==0){
            answer=s.charAt(s.length()/2-1)+""+s.charAt(s.length()/2);
        }else{
            answer=s.charAt(s.length()/2)+"";
        }
        return answer;
    }
}
```

* charAt(문자열 인덱스) : 해당하는 인덱스의 문자가 반환
